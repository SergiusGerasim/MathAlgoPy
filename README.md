# mathAlgoPy

Набор учебных ноутбуков и скриптов по вычислительным методам: нелинейные уравнения, ОДУ, ЧДУ, интерполяция и спектральные задачи.

(методы были реализованны в 2023-2024 годах, однако собраны в единое сборник и загруженны на гитхаб только в 2026)

## Файлы

- `nonlinear_root_finding_methods.ipynb`  
  Методы поиска корней нелинейных уравнений: regula falsi, модифицированный regula falsi, Ньютон, секущие и Стеффенсен; сравнение точности и числа итераций.

- `interpolation_newton_chebyshev_spline_methods.ipynb`  
  Интерполяция функций: полином Ньютона, равномерные и чебышевские сетки, кубические сплайны, оценка ошибок.

- `ode_euler_rk4_adams_methods.ipynb`  
Численное решение задач Коши для ОДУ/систем ОДУ методами Эйлера, Рунге-Кутты 4-го порядка и схемами Адамса.

- `gear4_method_demo.ipynb`  
  Демонстрационный ноутбук: вызов реализации метода Гира 4-го порядка и визуализация результатов.

- `gear4_method_impl.py`  
  Реализация метода Гира 4-го порядка и связанных вспомогательных функций для численного решения ОДУ.

- `qr_eigenvalue_methods.ipynb`  
  QR-разложение (Хаусхолдер), QR-алгоритм и вычисление собственных значений/векторов, сравнение с NumPy.

- `pde_finite_difference_thomas_method_demo.ipynb`  
  Разностные схемы для ЧДУ, решение трехдиагональных систем методом прогонки (Томаса), анализ порядка аппроксимации и граничных условий.

- `boffetta_osborne_scattering_method.ipynb`  
  Реализация метода Boffetta-Osborne для задачи рассеяния (NFT) и сравнение с аналитическими тестами.
